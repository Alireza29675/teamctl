# teamctl docs

Source for the docs site published at [teamctl.run](https://teamctl.run) (Cloudflare Pages).

## Local development

```bash
cd docs
npm install
npm run dev        # hot-reloading dev server on :4321
npm run build      # static site to docs/dist/
```

## Structure

- `src/content/docs/index.mdx` — landing hero.
- `getting-started.md` + `concepts/` + `reference/` + `guides/` + `adrs/` are Starlight-consumed markdown.
- `astro.config.mjs` — sidebar + Starlight theme config.

The CI workflow `.github/workflows/docs.yml` builds on push to `main` and deploys to Cloudflare Pages.

## Adding a page

1. Drop a `.md` or `.mdx` file under the appropriate directory.
2. Add a sidebar entry in `astro.config.mjs` unless the directory is autogenerated (`adrs/` is).
3. Commit. CI builds and deploys.
